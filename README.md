# --Ecommerce-backEnd--

# Intro

This is the Server side backend project made with node js .And  html template is used 

## Clone the Repo 

To clone the repo:-

```bash
~$ git clone "https://github.com/chhetri123/--Ecommerce-backEnd--.git"
```

## Installation 
1)
```bash
~$ cd --Ecommerce-backEnd--
```
2)
```bash
~$ npm install
```

## To Start
```bash
~$ npm start
```
